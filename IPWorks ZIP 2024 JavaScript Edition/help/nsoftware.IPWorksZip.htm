Introduction

Welcome to IPWorks Zip, a suite of easy, fast, and effective components that enable developers to rapidly add compression and decompression functionality to any application. The toolkit supports major compression/decompression and archiving standards, including Zip, Jar, Gzip, 7-Zip, Bzip2, ZCompress, and Tar. The IPWorks Zip components were built from the ground up, combining exceptional speed with an excellent data compression ratio.

Getting Started

IPWorks ZIP 2024 is delivered as a module which must first be installed prior to use. To install the module run the following command:

npm install ipworkszip-24.0.tar.gz

The module should now be installed and available for use. To use the module in your application first reference the package like so: const ipworkszip = require("@nsoftware/ipworkszip"); Instances of the classes listed below can then be created in code.

Included Classes

Bzip2 The Bzip2 component implements a bzip2 compressor and decompressor. It is compliant with bzip2 specification.
Gzip The Gzip component implements a gzip compressor and decompressor. It is compliant with RFC 1952 and compatible with the UNIX gzip and gunzip utilities.
Jar The Jar component may be used to create or decompress Java archives.
OfficeDoc The OfficeDoc component implements support for the Open XML Packaging Format used in Office 2007 documents.
SevenZip The SevenZip component implements a 7zip-compatible compressor and decompressor.
Tar The Tar component implements a tar archive utility, compatible with the UNIX tar and untar utilities. The component supports the old v7 tar format. The component also implements gzip compression and can be used to create or decompress .tar.gz archives.
ZCompress The ZCompress component implements an LZC compressor and decompressor. It is compatible with the UNIX compress utility.
Zip The Zip component implements a PKZip-compatible Zip compressor and decompressor.
ZipStream The ZipStream component is used to perform compression or decompression.
