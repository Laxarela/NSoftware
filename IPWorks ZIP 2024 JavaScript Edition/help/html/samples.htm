Sample Projects

The sample projects listed below demonstrate the usage of the IPWorks ZIP components in a simple, straightforward way. By default, the sample projects are installed in the demos subfolder of the installation directory.
Sample ProjectDescription
Create SevenZip

Shows how to create a SevenZip (.7z) archive.

Create Zip

Shows how to create a zip file.

Open SevenZip

Shows how to examine and extract the contents of a SevenZip (.7z) file.

Open Zip

Shows how to examine and extract the contents of a zip file.

NOTE: These projects are mostly simple demos and by no means complete applications. They are intended to illustrate the usage of the IPWorks ZIP components in a simple, straightforward way. What we are hoping to demonstrate is how simple it is to program with our tools. If you want to know more about them, or if you have questions, please visit www.nsoftware.com or email to support@nsoftware.com.
